Keeping the Server Running Forever

replit provides a free server for us to use, however if the server is not interacted with for too much time, the server will stop and the website won't run
To prevent this, we will be using uptimerobot which will ping the server once every five minutes.
First of all, sign up or log in to uptimerobot
Once you have entered, click on the 'Add New Monitor' button

In 'monitor type' select HTTP(s)
Enter any friendly name you want, although it should be descriptive
For the url, enter https://replit_projectname.replit_username.repl.co
And as for the monitoring interval, you can shorten or lengthen it, but I'd recommend keeping it at 5 minutes
When you're done, click 'Create Monitor'
